# PitsOfErrors

A terminal-based number guessing game that insults you for every wrong answer. Each wrong guess gives you a unique roast until you find the correct number.

## Features

Thousands of unique auto-generated insults

Hidden easter eggs for specific numbers

Colored terminal output using the rich library


## How to Run

Clone the repo and install dependencies:
```
git clone https://github.com/Rehana-Rahman/PitsOfErrors.git
cd PitsOfErrors
pip install -r requirements.txt
python main.py
```

## Easter Eggs >.<

Check eastereggs.txt for secret numbers that trigger custom messages.
